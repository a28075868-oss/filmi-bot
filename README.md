# Telegram Movie Bot 🎬

Telegram-бот для поиска информации о фильмах через OMDb API.

## Возможности

- 🔍 Поиск фильмов по названию
- 🖼️ Отображение постера фильма
- ⭐ Показ рейтинга IMDb
- 📝 Подробное описание фильма
- 🎬 Кнопка для поиска фильма онлайн
- 🎥 Кнопка для поиска трейлера на YouTube

## Требования

- Node.js версии 14 или выше
- npm или yarn
- Telegram Bot Token
- OMDb API Key

## Установка

### 1. Клонируйте репозиторий или создайте папку проекта

```bash
mkdir telegram-movie-bot
cd telegram-movie-bot
```

### 2. Установите зависимости

```bash
npm install
```

Эта команда установит следующие пакеты:
- `node-telegram-bot-api` - библиотека для работы с Telegram Bot API
- `axios` - HTTP-клиент для запросов к OMDb API
- `dotenv` - загрузка переменных окружения из .env файла

### 3. Получите необходимые API ключи

#### Telegram Bot Token:
1. Откройте Telegram и найдите [@BotFather](https://t.me/botfather)
2. Отправьте команду `/newbot`
3. Следуйте инструкциям для создания бота
4. Скопируйте полученный токен

#### OMDb API Key:
1. Перейдите на [OMDb API](http://www.omdbapi.com/apikey.aspx)
2. Выберите FREE план (1000 запросов в день)
3. Введите email и подтвердите его
4. Скопируйте полученный API ключ

### 4. Настройте переменные окружения

Создайте файл `.env` в корне проекта:

```bash
cp .env.example .env
```

Откройте `.env` и добавьте ваши ключи:

```env
TELEGRAM_BOT_TOKEN=ваш_telegram_bot_token
OMDB_API_KEY=ваш_omdb_api_key
```

### 5. Запустите бота

```bash
npm start
```

Вы должны увидеть сообщение: `🤖 Бот успешно запущен!`

## Использование

1. Найдите вашего бота в Telegram по username, который вы указали при создании
2. Отправьте команду `/start` для начала работы
3. Отправьте название фильма (на английском или русском языке)
4. Получите информацию о фильме с постером и кнопками

### Примеры запросов:

- `Inception`
- `The Matrix`
- `Interstellar`
- `Зеленая миля`
- `Матрица`

- `/start` - Начать работу с ботом

## Обработка ошибок

Бот включает обработку следующих ошибок:

- ❌ Фильм не найден
- ❌ Ошибка загрузки постера
- ❌ Ошибка API запроса
- ❌ Ошибка polling соединения

## Технические детали

- Использует **polling** для получения обновлений от Telegram
- Все строки запросов кодируются через `encodeURIComponent`
- HTML-форматирование для красивого отображения информации
- Inline-кнопки для быстрого доступа к дополнительным ресурсам

## Поддержка

Если у вас возникли проблемы:

1. Убедитесь, что все зависимости установлены: `npm install`
2. Проверьте правильность API ключей в `.env`
3. Убедитесь, что Node.js версии 14 или выше: `node --version`
4. Проверьте логи в консоли на наличие ошибок
